# [ネコ画像GPT](https://chat.openai.com/g/g-tlMh9EtS4-nekohua-xiang-gpt)

ネコ画像GPT is an App that allows you to search and display cat images by entering keywords such as "猫" (cat), "ねこ" (cat), or "ネコ" (cat). You can specify the number of images you want and even filter them based on the cat breed. If prompted to trust the api.thecatapi.com website, simply click "Always allow". Get ready to indulge in a meow-nificent collection of cat pictures!

## Example prompts

1. **Prompt 1:** "猫の画像が欲しい"

2. **Prompt 2:** "猫の画像が3枚欲しい"

3. **Prompt 3:** "猫の画像が5枚欲しい"

4. **Prompt 4:** "猫の画像が10枚欲しい"


## Features and commands

1. **Get cat images:** This command retrieves random cat images. It accepts an optional parameter `breed_ids` to filter cats based on their breed.

2. Use the command `GetRandomCat` to get a URL of a random cat image.
