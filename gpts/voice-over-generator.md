# [Voice Over Generator](https://chat.openai.com/g/g-R4H9Al3sl-voice-over-generator)

Create catchy audio scripts with Voice Over Generator! This app allows you to write scripts for various purposes, such as jingles for a rock radio station, podcast intros for a tech show, DJ drops for an EDM set, on-hold voicemails for a clinic, podcast promos for a true crime series, and funny liners for a morning show. Once you've written your script, Voice Over Generator instantly turns it into a voice over audio file that you can download. Get creative with your audio projects and make a lasting impression!

## Features and commands

1. **TextToSpeech** - This command turns a text script into a voice over audio file for download.

   Usage: To convert a text script into a voice over, use the `TextToSpeech` command followed by the text you want to convert. For example, `TextToSpeech [text]`.

   Example: `TextToSpeech "Welcome to our rock radio station, where the rhythm never stops!"`
