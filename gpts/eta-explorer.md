# [ETA Explorer](https://chat.openai.com/g/g-ZrBipXCBf-eta-explorer)

ETA Explorer is a chat-based app powered by ChatGPT that provides ETA (Estimated Time of Arrival) insights. Whether you want to know what ETA means, start your ETA journey, or learn about SBA loans, ETA Explorer has got you covered. With access to knowledge, it can provide valuable information and answers to your ETA-related questions. The app welcomes you with a prompt to explore the 2022 Search Fund Study. It also provides various tools like DALL-E for image generation and a browser tool for web browsing. Dive into the world of ETA exploration with ETA Explorer!

## Example prompts

1. **Prompt 1:** "What is ETA?"
2. **Prompt 2:** "How do I start my ETA Journey?"
3. **Prompt 3:** "What is SBA Loan?"
4. **Prompt 4:** "Is ETA right for me?"
5. **Prompt 5:** "Tell me more about the 2022 Search Fund Study."

## Features and commands

| Feature/Command | Description |
| --- | --- |
| `dalle` | This tool uses the DALL-E model for generating visual images based on input prompts. |
| `browser` | This tool allows you to browse webpages and retrieve information from the internet. |
