## Features and commands

| Feature/Command | Description |
| --- | --- |
| `generateDesigns` | This command allows you to get a list of Canva designs or templates to choose from based on a specific query. You can provide a description of the design you want to generate, such as "motivational cat poster," "wedding Instagram post," or "birthday card for my dog." The query should be a maximum of 140 characters. You also need to provide the locale, which is the 2-letter ISO language code and the 2-letter country code separated by a dash. |
