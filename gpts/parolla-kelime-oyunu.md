# [parolla - Kelime oyunu](https://chat.openai.com/g/g-oQzf4efes-parolla-kelime-oyunu)

parolla - Kelime oyunu is a fun and competitive word game app. Solve daily questions and join the competition. Create your own set of question and answers, and have fun solving the entertaining questions created by other players. Challenge yourself and expand your vocabulary while having a great time!

## Example prompts

1. **Prompt 1:** "I want to start the daily game."

2. **Prompt 2:** "Start the unlimited mode."

## Features and commands

| Feature/Command | Description |
| --- | --- |
| `get-questions` | This command allows you to retrieve the daily updated questions for the game. |
| `get-unlimited-questions` | This command retrieves different questions every time it is called for the unlimited mode of the game. |
